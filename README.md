# mint-lang-zsh
**mint-lang-zsh** is a Zsh plugin that provides arguments and options completions for `mint` command.

## Requirements
- Zsh
- Oh my Zsh

## Installation
```sh
git clone --depth=1 https://github.com/tamdaz/mint-lang-zsh ~/.oh-my-zsh/custom/plugins/mint
```
